### How to debug
    - port 8081 error
```zsh
$ rm -rf node_modules
$ npm clean cache
$ watchman watch-del-all
$ npm i
$ npm start -- --reset-cache
```
- gradle error
```zsh
$ chmod 755 android/gradlew 
```
